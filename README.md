Automação Selenium

Projeto de automação web utilizando Selenium no site da Farmácia Pague Menos.

## Funcionalidades

- Pesquisa de produtos
- Pesquisa com erro
- Seleção de categorias
- Adição ao carrinho
- Remoção de produtos
- Cadastro de CEP
- Validação de inconsistência na quantidade de produtos
- Ordenação de produtos por menor preço
